## Getting Started

```bash
# clone the project
git clone https://github.com/rcyj-FED/vue3-composition-admin.git

# enter the project directory
cd vue3-composition-admin

# install dependency
yarn # Or npm install

# develop
yarn start # OR npm run start
```
